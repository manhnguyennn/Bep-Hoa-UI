Mã lỗi Code Status Nội dung lỗi
#0 False Không có lỗi phát sinh, mã code hợp lệ và được áp dụng cho đơn hàng
#10x False Báo lỗi về chương trình Voucher
#100 False Voucher Code không tồn tại (Get từ DB theo code gửi lên không có dữ liệu)
#101 False Chương trình Voucher không tồn tại (Tham chiếu từ VoucherCode.VoucherId, có thể đã bị xoá dữ liệu)
#102 False Chương trình Voucher đã bị khoá hoặc huỷ nên code không sử dụng được
#103 False Chương trình Voucher bật check Đơn phải tối thiểu và giá trị đơn của khách chưa đạt giá trị này nên code không dùng được
#12x False Báo lỗi về Code
#120 False Chương trình Voucher đã hết hạn (So sánh thời gian hiện tại với Voucher.DateExpired và thêm điều kiện Voucher.IsExpired = true)
#121 False Chương trình phân phối tự động và Code chưa được phân phối thì không được dùng
#122 False Voucher Code đã bị vô hiệu hoá
#123 False Voucher Code đã bị sử dụng
#124 True Code freeship nhưng do chọn tự đến lấy và giá ship vốn đã = 0 nên trả về mã lỗi này
#125 False Code freeship khớp nhưng do đơn không đạt giá trị đơn tối thiểu
#126 True Mã giảm giá cho sp trong giỏ hàng nhưng giỏ hàng khách không có món ăn khớp với danh sách món ăn được add vào voucher nên giá giảm = 0
#127 False Khách đã sử dụng chương trình 1 lần với mã code khác rồi nên khi dùng 1 mã code trên cùng 1 chương trình đã sử dụng thì sẽ không được dùng nữa
#128 False Voucher Code không đúng 6 ký tự
#129 False Voucher Code FREESHIP nhưng có bật giảm từng sản phẩm nhưng trong giỏ hàng không có món đó nên k dùng được
